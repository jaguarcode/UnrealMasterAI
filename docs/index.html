5-Minute Setup

Get Started in Three Commands

From zero to controlling Unreal Engine with natural language.

1

Install & Build the MCP Server

Clone the repo and build the TypeScript MCP bridge server. Requires Node.js 20+.

2

Configure Claude Code

Add the server to .claude/mcp.json. Claude Code auto-discovers all 183 tools on startup.

3

Enable the UE Plugin

Copy ue-plugin/ into your project's Plugins/ directory. The plugin auto-connects to the MCP server on editor launch.

4

Start Building

Open Claude Code and describe what you want: "Add a health system to BP_Enemy". The agent handles the rest.

# 1. Clone & build git clone https://github.com/jaguarcode/UnrealMasterAI cd UnrealMasterAI/mcp-server npm install && npm run build # 2. Add to .claude/mcp.json { "mcpServers": { "unreal-master": { "command": "node", "args": ["mcp-server/dist/index.js"], "env": { "UE_WS_PORT": "9877" } } } } # 3. Copy plugin to your UE project cp -r ue-plugin/ YourProject/Plugins/UnrealMasterAgent # 4. Verify connection claude "ping the editor" { "status": "ok", "ueVersion": "5.4.x" }

How It Works

From Natural Language to Engine Changes

Describe your intent. Claude orchestrates the tools. Unreal executes in real time.

1

Describe Intent

"Add a PrintString node to BP_TestActor after BeginPlay, set the message to Hello World."

2

Serialize & Plan

Claude serializes the Blueprint to JSON AST, analyzes the graph, and plans the optimal tool sequence.

3

Execute Changes

Creates nodes, connects exec pins, sets values — all dispatched to the GameThread via C++ engine APIs.

4

Compile & Heal

Triggers Live Coding. If compilation fails, parses errors, applies fix, and retries up to 3 times automatically.

Trust & Safety

3-Tier Safety System

Every tool classified by risk. Destructive operations require explicit human approval.

Safe

Read-only queries. List assets, inspect Blueprints, query project structure. No approval needed.

Warn

Recoverable mutations. Create assets, spawn actors, modify properties. Logged and auto-approved.

Dangerous

Destructive operations. Delete assets, cook content, exec console commands. Requires Slate dialog approval with 60s timeout.
